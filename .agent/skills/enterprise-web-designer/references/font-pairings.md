# Font Pairings Theo Aesthetic Direction

## Luxury / Premium
- Display: `Playfair Display` (serif, elegant)
- Body: `DM Sans` hoặc `Jost`
- Accent: `Cormorant Garamond` (italic captions)

## Modern Corporate
- Display: `Syne` hoặc `Plus Jakarta Sans`
- Body: `DM Sans`
- Accent: `Space Grotesk` (nhưng dùng tiết kiệm)

## Organic / Warm
- Display: `Fraunces` (variable, personality)
- Body: `Nunito` hoặc `Lato`
- Accent: `Caveat` cho handwritten feel

## Bold / Brutalist
- Display: `Bebas Neue` hoặc `Anton`
- Body: `IBM Plex Sans`
- Accent: `Azeret Mono` (monospace)

## Tech / Futurist
- Display: `Orbitron` hoặc `Exo 2`
- Body: `IBM Plex Sans`
- Accent: `JetBrains Mono`

## Editorial / Magazine
- Display: `Instrument Serif` hoặc `Libre Baskerville`
- Body: `Source Serif 4`
- Accent: `Barlow Condensed`

## Cách Load (Google Fonts):
```html
<link rel="preconnect" href="https://fonts.googleapis.com">
<link href="https://fonts.googleapis.com/css2?family=Playfair+Display:wght@400;700&family=DM+Sans:wght@300;400;500&display=swap" rel="stylesheet">
```
